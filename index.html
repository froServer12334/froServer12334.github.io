About

Basically this is a moded server and you goal is to destroy as many players as possible. You are free to do anything you want to so long as it doesn’t break the rules. I recommend setting up a base of operations to craft advanced armour and weapons. LagGoggles has been installed on this server to help you and the admin monitor lag. The config and mod files can be found at the bottom of the page. Also, it’s free to play.

Installation and troubleshooting:

I recommend that you set a lot at 6 Gb of ram to Minecraft and have a dedicated GPU. You could get away with 4Gb and using integrated graphics but it’s not recommended. I also recommend that you use 64-bit Java. You will also need to make sure that your mods and config folders match. I recommend that you use Multi Mc, it allows you to keep your installations separate. Note: If you bought Minecraft with a Microsoft account, you wont be able to use Multi Mc (they are working on migrating accounts so you should be able to use it in the future).

Note: For the tutorials below make sure you use Minecraft version 1.12.2, allocate 6Gb of RAM, install Forge version 1.12.2 - 14.23.5.2854 and use the mod and config files provided below The config file is not 100% required as your config files will update automatically when connected to the server, you will just need to restart your game.

  • “Missing Mod Error”: Redownload the mod file and put the mods into your Minecraft folder.
  • “Mod config error”: If there is an option presented on the screen to pull from the server select it then restart your game. If not, then delete your config folder, download the on from the link about and paste it in.
